### 🎯 3. Main Point
- 전국적으로 공공심야약국을 운영중에 있으며 전주시에는 2곳이 위치하고 있음. 타지역의 경우 이용률이 점차 증가하는 추세를 보였고, 전주시 인구 대비 추가 공공심야약국이 필요하다는 문제의식에서 프로젝트를 시작함.  
- 전주시의 다양한 데이터를 활용해 군집분석, GIS분석을 진행했고 최적 입지 선정을 진행했음.
